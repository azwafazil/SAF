# SAF MCP Roadmap

## v0.1 MVP
- Safe local file access
- CSV/XLSX/SAV reading
- Data dictionary
- Descriptives, frequencies, reliability, t-test, ANOVA, correlation, chi-square, OLS
- SPSS syntax draft generator
- Markdown report export

## v0.2 Student Assignment Mode
- Research question to test recommendation
- Null/alternative hypothesis generator
- APA 7 explanation generator
- SPSS menu-path helper
- Assumption checker per test

## v0.3 Survey Scale Mode
- Reverse-coded items
- Composite score creation
- Corrected item-total correlation
- Alpha-if-item-deleted table
- Scale report

## v0.4 Factor Analysis Mode
- KMO
- Bartlett's test
- EFA extraction
- Varimax/oblimin rotation
- Loading table

## v0.5 Thesis Mode
- Full reproducible recipe pipeline
- Export to DOCX/XLSX/Markdown
- Audit ledger
- Versioned outputs

## v1.0 SPSS Companion
- Strong `.sav` metadata preservation
- SPSS-like tables
- Test chooser
- Report generator
- Security hardening
- MCP prompts/resources
